证明 $\lim\limits_ {n\to +\infty} \sqrt[n]{n} = 1$

令 $t=\sqrt[n]{n} -1$,则$n=(1+t)^n$
二项式展开得$n=(1+t)^n > \binom{n}{2}t^2 = \frac{n(n-1)t^2}{2}$
$\therefore t< \sqrt[2]{\frac{2}{n-1}} $
则$0<\sqrt[n]{n}-1< \sqrt[2]{\frac{2}{n-1}}$
显然$\lim\limits_ {n\to +\infty} \sqrt[2] { \frac{2}{n-1}} =0$
由夹逼定理得$\lim\limits_ {n\to +\infty} \sqrt[n]{n} -1 = 0$
则$\lim\limits_{n\to +\infty} \sqrt[n]{n} = 1$

然而明明用洛必达可以轻松解决的事。。
$\sqrt[n]{n}=e^{\frac{1}{n} * \ln n}$
$\lim\limits_ {n\to +\infty} {\frac{\ln n}{n}}$
$=\lim\limits_ {n\to +\infty} {\frac{\frac{1}{n}}{1}} =0$
$e^0 = 1,\therefore \lim\limits_ {n\to +\infty}{\sqrt[n]{n}}=1$

书上的提示是证明对任意正数$\xi >0,\lim\limits_{n\to \infty}{\frac{n}{(1+\xi)^n}} =0$,再证$\exists N\in N$,使得$n>N$时，$\frac{n}{(1+\xi)^n}<1$.
然而并没有搞懂这个提示的目的欸。。。我好菜啊
